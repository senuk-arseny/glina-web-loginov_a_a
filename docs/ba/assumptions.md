# assumptions.md — Гончарная мастерская «Глина»

## Принимается по умолчанию (MVP)
- Регистрация/авторизация клиента — простая (имя + телефон, без SMS-подтверждения на учебном проекте).
- «Поздняя» отмена — менее чем за 2 часа до начала. Санкций за неё в MVP нет, только фиксация факта в истории клиента.
- Прокат инструментов/фартука — платная опция, доступность ограничена фондом комплектов.
- Оценка мастера — необязательна, доступна после факта посещения, видна другим клиентам как публичный средний рейтинг.
- Лист ожидания на заполненный слот — входит в MVP.
- «Постоянный клиент» — статус-метка при 3+ завершённых посещениях, без скидочной логики.
- Уведомления — in-app (в приложении): подтверждение записи, напоминание перед занятием, отмена мастерской.
- Один клиент может иметь неограниченное число активных записей.
- Оплата в приложении отсутствует — бронирование бесплатно резервирует место, оплата очно в мастерской.
- Лимит участников зависит от программы: новичковая — 6 чел., «круг» — до 10.
- Клиентское приложение — только потребитель API; слоты, программы, мастеров не создаёт/не редактирует.
- Горизонт отображения слотов — 7 дней по умолчанию, с фильтром на более поздние даты.

## Вне scope (текущая поставка)
- Онлайн-оплата в приложении.
- Интерфейсы мастера и администратора/владельца.
- Логика скидок/бонусов для постоянных клиентов.
- Штрафы/блокировки за поздние отмены и no-show.
- Создание/редактирование слотов, программ, мастеров со стороны клиента.
- Само действие отмены занятия мастерской (форс-мажор) — выполняется в существующей инфраструктуре, клиент только видит результат.

## Future phase

- Онлайн-оплата.
- Возможные санкции за систематические no-show.
