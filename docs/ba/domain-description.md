# domain-description.md — Гончарная мастерская «Глина»

## Actors

### Client (в скоупе MVP)
- Просматривает расписание/слоты на ближайшие 7 дней
- Бронирует место на слот, выбирает опцию проката
- Отменяет свою бронь, встаёт в лист ожидания
- Оценивает мастера после посещения занятия
- Просматривает профиль, историю записей, статус «постоянный клиент»
- Получает уведомления (подтверждение, напоминание, отмена мастерской)

### Master (вне скоупа текущей поставки)
- Ведёт занятия. Интерфейс мастера уже существует в инфраструктуре.

### Admin/Owner (вне скоупа текущей поставки)
- Управляет расписанием вручную при сбоях. Интерфейс уже существует в инфраструктуре.

## Entities

| Entity | Назначение | Ключевые поля | Editable клиентом? |
|---|---|---|---|
| Program | Тип занятия | id, name, description, maxParticipants | read-only |
| Slot | Конкретное занятие в расписании | id, programId, masterId, startTime, duration, status | read-only |
| Master | Мастер, ведущий занятие | id, name, avgRating | read-only |
| Booking | Бронь клиента на слот | id, clientId, slotId, equipmentRentalRequested, status | editable (создание/отмена) |
| WaitlistEntry | Позиция в листе ожидания | id, clientId, slotId, status, createdAt | editable (создание/удаление) |
| EquipmentRental | Доступность проката на слот | slotId, availableSets, pricePerSet | read-only |
| Review | Оценка мастера клиентом | id, bookingId, masterId, rating, comment | editable (создание, разово) |
| Client | Профиль пользователя | id, name, phone, isRegular, visitedCount | частично editable |

## Scope Boundaries

**Входит в MVP:** просмотр расписания, бронирование, отмена, лист ожидания, прокат, оценка мастера, профиль, уведомления.

**Уже существует на backend:** формирование расписания, атомарная проверка мест, интерфейсы мастера/админа, само действие отмены занятия мастерской.

**Вне scope:** онлайн-оплата, скидочные механики, санкции за поздние отмены.

## Glossary

- **Слот** — конкретное занятие в расписании.
- **Программа** — тип занятия: новичковая лепка (лимит 6) или круг (лимит 10).
- **Прокат** — платная опция аренды инструментов, ограниченный фонд.
- **Лист ожидания** — очередь на заполненный слот.
- **Постоянный клиент** — статус при 3+ завершённых посещениях.
- **Форс-мажор** — отмена занятия мастерской, бронь получает статус «Отменено мастерской».
