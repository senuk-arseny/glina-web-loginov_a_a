# screen-specs.md — Гончарная мастерская «Глина»

## Список занятий
Фильтр по дате (с/по), список карточек (программа, время, мастер, места, рейтинг). Empty state «Пока нет доступных занятий». Error → «Не удалось загрузить расписание».

## Детали занятия
Программа, мастер+рейтинг, время, места, прокат. Кнопка «Записаться» → «В лист ожидания», если мест нет. Если уже забронировано этим клиентом — баннер вместо кнопки. Если отменено мастерской — баннер с причиной.

## Мои записи
Секции по статусу брони + отдельная секция «Лист ожидания» с позицией в очереди. Кнопки «Отменить», «Оценить мастера» (для завершённых). Демо-хелперы для тестирования сценариев без реального времени/бэкенда.

## Профиль
Имя, контакт, число посещений, статус «постоянный клиент», число поздних отмен.

## Уведомления
Список карточек по типу (подтверждение/напоминание/отмена/лист ожидания), сортировка по дате, бейдж непрочитанных в навигации.

## Главная (лендинг)
Хиро с фото и кольцами, «как это работает», программы, мастера, CTA-секция. Фото — реальные файлы-плейсхолдеры в `public/images/` (8 штук: хиро, 2 программы, 4 мастера, фон CTA), подставлены через `<img src="...">` с явным путём на каждого мастера — не общий плейсхолдер на всех четверых. Список путей и что на них должно быть — см. `HomePage.tsx`, константы `MASTER_PHOTOS`/`PROGRAM_PHOTOS`/`HERO_PHOTO`/`CTA_PHOTO`.

## Вход/регистрация
Карточка-бирка: имя, телефон, кнопка «Войти в мастерскую».
