# Feature 01: Список занятий с фильтром по дате

## Цель
Клиент должен видеть расписание мастерской на ближайшие 7 дней и уметь отфильтровать его по произвольному диапазону дат — это основной "вход" в приложение, замена Instagram-директу и ежедневнику.

## Требования
- FR-01, FR-02 (`docs/ba/functional-requirements.md`)
- User story 1, 2 (`docs/ba/user-stories.md`)
- Screen: «Список занятий» (`docs/ba/screen-registry.md`, `screen-specs.md`)

## Реализация
- `src/pages/SlotListPage.tsx` — фильтр по датам (input[type=date] × 2) + список карточек
- `src/mock/api.ts` → `listSlots(dateFrom?, dateTo?)` — дефолт 7 дней, либо явный диапазон
- `src/utils/format.ts` → `parseDateInputStart/End`, `formatDateTimeWithWeekday` — корректный парсинг границ дат и единый формат времени

## Промпты (сокращённо)
- Запрос на реализацию мок-API и слоя данных, повторяющего контракт `openapi.yaml`, с бизнес-логикой на стороне "бэкенда"
- Отдельный промпт на компонент списка занятий с фильтром и empty state
- Уточняющий промпт на исправление off-by-one в фильтре дат (конец диапазона не включал последний день) — см. `docs/bugs/bug-02-date-filter-off-by-one.md`

## Ручная проверка
- TC-01, TC-02, TC-03 (`docs/ba/test-cases.md`) — пройдены
- Проверено вручную: фильтр "с 4 по 6 июля" показывает все три дня, включая последний
